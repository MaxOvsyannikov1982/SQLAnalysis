##  SQL E-commerce Analytics Project
Проект имитирует работу аналитика в интернет-магазине. Включает в себя проектирование схемы базы данных и глубокий анализ ключевых бизнес-метрик.

### Структура файлов
*   `01_schema.sql` — Архитектура таблиц и связей (DDL).
*   `02_automation.sql` — Триггеры для автоматического обновления данных профиля.
*   `03_seed_data.sql` — Наполнение данными и скрипты очистки (Data Cleaning).
*   `04_analytics_queries.sql` — Бизнес-аналитика и сложные запросы.

### Реализованные метрики
*   **LTV (Lifetime Value)** — расчет общей ценности каждого клиента.
*   **Cohort Analysis** — анализ удержания клиентов по месяцам регистрации.
*   **Category Ranking** — сегментация товаров по прибыльности (DENSE_RANK).
*   **Running Total** — расчет динамики выручки с помощью оконных функций.

### Стек технологий
*   SQL (PostgreSQL)
*   DDL/DML
*   CTE (Common Table Expressions)
*   Window Functions

### Как запустить
1. Создайте базу данных в PostgreSQL.
2. Выполните SQL-файлы строго по порядку их нумерации.
3. Ознакомьтесь с результатами в логах или через BI-инструмент.
