1. Заголовок и Описание
# SQL E-commerce Analytics Project
Проект имитирует работу аналитика в интернет-магазине. Включает в себя полный цикл: от проектирования схемы базы данных до глубокого анализа метрик (LTV, Retention).
2. Структура проекта (File Structure)
01_schema.sql — Архитектура таблиц и связей.
02_automation.sql — Триггеры для автоматического обновления данных профиля.
03_seed_data.sql — Наполнение данными и скрипты очистки (Data Cleaning).
04_analytics_queries.sql — Бизнес-метрики (LTV, Когорты, Оконные функции).
3. Реализованные метрики
LTV (Lifetime Value) — общая ценность каждого клиента.
Cohort Analysis — удержание клиентов по месяцам регистрации.
Category Ranking — сегментация товаров по прибыльности с использованием DENSE_RANK.
Running Total — динамика выручки в реальном времени.
4. Как запустить
Создайте базу данных в PostgreSQL.
Выполните файлы по порядку номеров.
Ознакомьтесь с результатами в логах или через BI-инструмент.
5. Стек: SQL (PostgreSQL), DDL/DML, CTE (WITH), Window Functions.
